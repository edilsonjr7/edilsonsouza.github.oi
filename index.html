Array length

A propriedade length em arrays é usada para determinar o número de elementos presentes em um array. Em outras palavras, ela retorna o tamanho do array. A sintaxe é bastante simples: basta acessar essa propriedade pelo nome do array seguido de .length.

Sintaxe:

array.length

Exemplos:

const frutas = ["maçã", "banana", "laranja"];
console.log(frutas.length); // Saída: 3

const vazio = [];
console.log(vazio.length); // Saída: 0

Método toString()

A função toString() de um array em JavaScript converte todos os elementos do array em uma string, separados por vírgulas. Essa função é útil quando você quer apresentar os dados de um array como texto.

Sintaxe:

array.toString()

Exemplos:

const numeros = [1, 2, 3, 4, 5];
const texto = numeros.toString();
console.log(texto); // Saída: "1,2,3,4,5"

const misto = ["maçã", 42, true, null];
const textoMisto = misto.toString();
console.log(textoMisto); // Saída: "maçã,42,true,"

Método at()

É uma forma de acessarmos elementos no array ou string, utilizando índices positivos ou negativos.

Sintaxe:

array.at(índice)

Exemplos:

const frutas = ['maçã', 'banana', 'laranja', 'uva'];

console.log(frutas.at(1));    // 'banana'
console.log(frutas.at(-1));   // 'uva'
console.log(frutas.at(-2));   // 'laranja'
console.log(frutas.at(10));   // undefined

const texto = "JavaScript";
console.log(texto.at(-1));  // 't'
console.log(texto.at(3));   // 'a'

// Comparação tradicional
const array = [10, 20, 30, 40, 50];
const ultimo = array[array.length - 1]; // 50
const ultimoComAt = array.at(-1); // 50

// Curiosidade
const letras = ['a', 'b', 'c', 'd'];
console.log(letras[-1]); // undefined

Método join()

O método join() transforma todos os elementos de um array em uma única string. Por padrão, os elementos são separados por vírgulas, mas é possível escolher outro separador.

Sintaxe:

array.join(separador)

Exemplos:

let frutas = ["Maçã", "Banana", "Laranja"];
console.log(frutas.join());       // "Maçã,Banana,Laranja"
console.log(frutas.join(" - "));  // "Maçã - Banana - Laranja"

Método pop()

O método pop() é uma função em JavaScript usada em arrays (que são listas de elementos, como números, strings, objetos, etc.). Ele tem uma função muito simples: remover o último item de um array e retornar esse item.

Como funciona?

  1. Você começa com um array de frutas: ["Maçã", "Banana", "Laranja"]
  2. Ao chamar frutas.pop(), o método remove o último item do array, que é "Laranja".
  3. O método pop() retorna "Laranja", então você pode usar esse valor ou armazená-lo em uma variável.
  4. Após o pop(), o array original agora é: ["Maçã", "Banana"]

Características:

  • Modifica o array original.
  • Retorna o item removido.
  • Remove sempre o último item.

Exemplo:

let frutas = ["Maçã", "Banana", "Laranja"];
let ultima = frutas.pop();
console.log(ultima); // "Laranja"
console.log(frutas); // ["Maçã", "Banana"]

Método push()

O método push() é usado para adicionar um ou mais elementos ao final de um array. Ele também retorna o novo comprimento do array após a adição.

Sintaxe:

array.push(elemento1, elemento2, ...)

Exemplos:

let frutas = ["Maçã", "Banana"];
frutas.push("Laranja");
console.log(frutas); // ["Maçã", "Banana", "Laranja"]

let numeros = [1, 2, 3];
numeros.push(4, 5);
console.log(numeros); // [1, 2, 3, 4, 5]

let letras = ["a", "b"];
let novoTamanho = letras.push("c");
console.log(novoTamanho); // 3
console.log(letras); // ["a", "b", "c"]

Resumo:

  • Adiciona elementos ao final do array.
  • Modifica o array original.
  • Retorna o novo comprimento do array.
