Array length em JavaScript

A propriedade length em arrays é usada para determinar o número de elementos presentes em um array. Em outras palavras, ela retorna o tamanho do array.

Sintaxe:

array.length

Exemplos:

Exemplo 1: Verificando o tamanho de um array simples

const frutas = ["maçã", "banana", "laranja"];
console.log(frutas.length); // Saída: 3

Exemplo 2: Usando length em um array vazio

const vazio = [];
console.log(vazio.length); // Saída: 0

Método toString() em JavaScript

O método toString() de um array converte todos os elementos do array em uma string, separados por vírgulas. Ele é útil quando se deseja apresentar os dados do array como texto.

Sintaxe:

array.toString()

Exemplos:

Exemplo 1: Convertendo um array de números para string

const numeros = [1, 2, 3, 4, 5];
const texto = numeros.toString();
console.log(texto); // Saída: "1,2,3,4,5"

Exemplo 2: Array com tipos mistos

const misto = ["maçã", 42, true, null];
const textoMisto = misto.toString();
console.log(textoMisto); // Saída: "maçã,42,true,"

Método at() em JavaScript

O método at() permite acessar elementos de um array ou string utilizando índices positivos ou negativos.

Sintaxe:

array.at(índice)

Exemplos com Arrays:

const frutas = ['maçã', 'banana', 'laranja', 'uva'];

console.log(frutas.at(1));    // 'banana'
console.log(frutas.at(-1));   // 'uva'
console.log(frutas.at(-2));   // 'laranja'
console.log(frutas.at(10));   // undefined

Exemplos com Strings:

const texto = "JavaScript";

console.log(texto.at(-1));  // 't'
console.log(texto.at(3));   // 'a'

Comparação com forma tradicional:

const array = [10, 20, 30, 40, 50];
const ultimo = array[array.length - 1]; // 50

Com at(), fica mais simples:

const ultimo = array.at(-1); // 50

Curiosidade:

Usar colchetes [] com índice negativo não funciona:

const array = ['a', 'b', 'c', 'd'];
console.log(array[-1]); // undefined
console.log(array[-2]); // undefined

Método join() em JavaScript

O método join() transforma os elementos de um array em uma única string, separados por um delimitador especificado.

Sintaxe

array.join(separador);

Exemplos

let frutas = ["Maçã", "Banana", "Laranja"];
console.log(frutas.join());       // "Maçã,Banana,Laranja"
console.log(frutas.join(" - "));  // "Maçã - Banana - Laranja"

Método pop() em JavaScript

O método pop() remove o último item de um array e retorna esse item.

Exemplo:

let frutas = ["Maçã", "Banana", "Laranja"];
let ultimaFruta = frutas.pop();
console.log(ultimaFruta); // "Laranja"
console.log(frutas);      // ["Maçã", "Banana"]

Resumo:

  • Modifica o array original.
  • Retorna o item removido.
  • Remove apenas o último item.

Método push() em JavaScript

O método push() é usado para adicionar um ou mais elementos ao final de um array. Ele também retorna o novo comprimento (tamanho) do array após a adição.

Sintaxe:

array.push(elemento1, elemento2, ...);

Exemplos:

Adicionando um item ao final de um array

let frutas = ["Maçã", "Banana"];
frutas.push("Laranja");
console.log(frutas); // ["Maçã", "Banana", "Laranja"]

Adicionando múltiplos itens

let numeros = [1, 2, 3];
numeros.push(4, 5);
console.log(numeros); // [1, 2, 3, 4, 5]

Verificando o novo tamanho do array

let letras = ["a", "b"];
let novoTamanho = letras.push("c");
console.log(novoTamanho); // 3
console.log(letras); // ["a", "b", "c"]

Resumo:

  • Adiciona elementos ao final do array.
  • Modifica o array original.
  • Retorna o novo comprimento do array.
